# Tasty-pl

Tasty-pl is a Spring Boot backend app for food delivery that works just like a well known website Pyszne.pl

<h1>
Technologies used:
</h1>
<ul>
  <li>Java</li>
  <li>Spring Boot</li>
  <li>Spring Security with JWT</li>
  <li>JPA + Hibernate</li>
  <li>MySQL</li>
  <li>JUnit + Mockito</li>
</ul>
